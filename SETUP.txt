## 📋 What's Included

This package contains everything you need to run the GitHub Profile README Generator:

- `app.py` - Main Streamlit application
- `requirements.txt` - Required Python packages
- `README.md` - This file with instructions

---

## 🚀 Quick Setup (For Your Friend)

### Step 1: Extract Files
Extract all files to a folder (e.g., `readme-generator`)

### Step 2: Install Python
Make sure you have Python 3.8+ installed on your computer

### Step 3: Install Dependencies
Open terminal/command prompt and run:
```bash
pip install -r requirements.txt
```

### Step 4: Run the Application
```bash
streamlit run app.py --server.headless true
```

The app will open in your browser at `http://localhost:8501`

---

## 🎯 How to Use

1. **Upload Profile Picture**: Use the drag & drop uploader in the sidebar
2. **Fill Your Information**: 
   - Name and GitHub username (required)
   - Bio, skills, social links (optional)
3. **Generate README**: Click "Generate README" button
4. **Download**: Click "Download README.md" to save your file
5. **Use on GitHub**: 
   - Create a repository named `yourusername/yourusername`
   - Upload the generated README.md file
